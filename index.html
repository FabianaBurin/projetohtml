O que é HTML?

Voltar

O HTML (Hypertext Markup Language) é uma linguagem de MARCAÇÃO utilizada na estruturação de páginas web.

Inspetor de elementos

Voltar

O inspetor de elementos é uma ferramenta oferecida pelo Google Chrome, utilizada para inspencionar os elementos da página, é uma ferramenta em que o usuário consegue visualizar o código HTML, entre outras coisas.

Estrutura básica do HTML

Voltar

A estrutura básica do HML é composta basicamente por: HTML, head e body.

A tag HTML <html></html> delimita a área de todo o HTML.

Na tag head <head></head> é definido o cabeçalho do documento, não é visível no navegador mas contém instruções sobre seu conteúdo e comportamento.

Já a tag body <body></body> é o espaço onde vai estar o conteúdo visual da página.

Essa é uma estrutura básica para se iniciar um texto com HTML!

Sobre tags

Voltar

Existem vários tipos de tags dentro do HTML, cada uma, com uma função como por exemplo:

A tag i <i></i> deixa o texto em ITÁLICO.

A tag b <b></b> deixa o texto em NEGRITO.

A tag strong <strong></strong> deixa o texto MARCADO.

Claro que para o HTML temos uma variedade enorme de tags para serem utilizadas, essas foram as básicas, mas na internet tem uma variedade grande de tags para serem utilizados no seu HTML.

Mas existem tags que não precisam de fechamento (</atributo>), pois pode ser inseridos atributos dentro da tag, como por exemplo:

<input (+atributo da tag)>

<img (+nome ou link da imagem)>

Aqui foi aprendido sobre as tags do HTML, para cada ação que queremos fazendo dentro do HTML, temos tags para ser inseridas, para que fique do jeito que queromos.

Atibutos das tags

Voltar

Os atributos são informações que colocamos dentro da tag, para que tenha um determinado comportamento.

Existem atributos globais (funcionam em todas as tags) e também tem os atributos específicos (que são direcionados para tags específicas).

Textos

Voltar

Para inserir textos no HTML temos as opções de título, subtítulo e parágrafo.

Para pagráfo é utilizado a tag p:

<p></p>

Já para títulos e subtítulos é utilizada de h1 à h6

<h1></h1> Maior e mais relevante.

<h2></h2>

<h3></h3>

<h4></h4>

<h5></h5>

<h6></h6> Menor e menos relevante.

Para destacar citação dentro da tag, deve ser inserida a tag blockquote:

<blockquote></blockquote>

Listas ordenadas e listas não ordenada

Voltar

No HMTL, temos as listas que podem ser odenadas (por número), ou não ordenada (por bolinha).

Em listas ordenadas utilizamos a tag 'ol' e os itens dessa lista, em cada linha deve estar incluída a tag 'li'

<ol>

<li> ITEM 1 </li>

<li> ITEM 2 </li>

<li> ITEM 3 </li>

</ol>

Já as listas não ordenadas, utilizamos a tag 'ul' e os itens da lista também, em cada linha tem que incluir a tag 'li'

<ul>

<li> ITEM 1 </li>

<li> ITEM 2 </li>

<li> ITEM 3 </li>

</ul>

No item de listas ordenadas e não ordenadas, vemos toda a estrutura para se incluir uma lista no HTML, que elas podem ser feitas de forma ordenada ou de forma não ordenada.

Links

Voltar

Os links dentro do HTML, são utilizados para direcionar o usuário para outras páginas ou para partes da mesma página e para isso é utilizado a tag 'a':

<a></a>

Sempre é inserido os atributos dentro da tag, para que seja direcionado para onde deseja.

No exemplo a seguir será mostrado como inserir um link que direicione você para um link na mesma aba em que você esta navegando:

<a link do site > O nome que irá aparecer para o usuário </a>

Ao clicar no link, você irá abrir na mesma aba outro site.

Também tem a opção de fazer um link abrir em outra aba:

<a link do site target="_blank" > O nome que irá aparecer para o usuário </a>

O atributo target direciona o usuário para uma nova aba.

Também podemos mostrar uma mensagem enquanto o usuário passa o mouse no link, que é utilizado o atributo title="Mensagem pequena que você deseja que apareça"
